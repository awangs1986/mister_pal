## Why not a “modify OK, commercial forbidden” license?

That style of license (e.g. CC BY-NC, PolyForm Noncommercial) **cannot** replace GPL-3.0
for code derived from SDLPAL and other GPL MiSTer cores. Doing so would violate upstream
license terms.

If you need a non-commercial policy for **separately authored content that is not a
GPL derivative** (e.g. your own docs or artwork with no upstream GPL code), keep that
content in a separate path/repo under a NC license. It is **not** applied to the engine
or FPGA/ARM hybrid core in this repository.
